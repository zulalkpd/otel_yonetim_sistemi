**Otel Yönetim Sistemi**

Bu proje, C programlama dili kullanılarak geliştirilmiş, dosya tabanlı çalışan temel bir yönetim sistemidir.. Müşteri kayıtlarını dijital ortamda tutmak ve otel verilerini analiz etmek amacıyla tasarlanmıştır.

<Proje Özellikleri>

Program şu temel işlevleri yerine getirmektedir:
* **Dinamik Kayıt Sistemi:** Müşteri bilgilerini alarak `otel.txt` dosyasına kalıcı olarak kaydeder.
* **Otomatik Ücret Hesaplama:** Konaklama gecesi üzerinden ücretlendirme yapar.
* **Akıllı Kategorizasyon:** Yaş sınırına göre müşterileri "Çocuk" veya "Yetişkin" olarak sınıflandırır.
* **Finansal & İstatistiksel Raporlama:** * Toplam müşteri sayısını takip eder.
    * Otelin toplam gelirini (ciro) hesaplar.
    * Müşteri geri bildirimlerinden puan ortalamasını çıkarır.
* **Veri Yönetimi:** Tek tuşla tüm kayıtlı verileri güvenli bir şekilde temizleme imkanı sunar.

<Teknik Detaylar>

* **Programlama Dili:** C
* **Veri Saklama:** Standart Dosya İşlemleri (txt üzerinden `fprintf`, `fscanf`)
* **Kontrol Yapıları:** Döngüler, Karar Yapıları, Fonksiyonlar

<Kullanım Talimatı>

1. Projeyi bir C derleyicisi ile açın.
2. Kodu derleyin ve çalıştırın.
3. Menü üzerinden yapmak istediğiniz işlemi (1, 2, 3 veya 0) seçerek ilerleyin.
4. Kaydedilen veriler programın bulunduğu klasörde otomatik olarak oluşturulan `otel.txt` dosyasında saklanacaktır.
